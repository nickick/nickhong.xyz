# nickhong.xyz
This is a static portfolio site for [@pepperonick](https://twitter.com/pepperonick).

## Packages used for this site:
* React.js
* Next.js
* Material UI
* Framer Motion
* Vercel.js for deployments

## Features remaining:
* Footer section
* Contact section
* Additional project cards
* Active sections for navigation

## To run:
```
npm install
npm run dev
```
